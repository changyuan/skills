# 写作前检查

> **触发时机**：开始写新文章前
> **目标**：确保选题正确，避免方向性错误

---

## 一、目标用户确认 ⭐⭐⭐⭐⭐

### 检查项

```
□ 已确定目标用户 ≥3 类？（必填）
□ 已阅读 user-personas.md？
□ 明确每类用户的：认知结构/技能基础/资源约束？
□ 包含小白用户？（必须）
```

### 成功案例对照

| 维度 | ✅ 成功案例 | ❌ 失败案例 |
|------|------------|------------|
| 用户覆盖 | 同时覆盖小白/进阶/资深用户 | 只写给单一用户群体 |
| 触达策略 | 每类用户有明确触达渠道 | 只有模糊的"目标用户" |
| 需求理解 | 精准描述用户4层需求 | 只有宽泛的"帮助用户" |

### 问题诊断

```
用户不清晰 → 暂停写作，先读 user-personas.md
用户<3类 → 扩展用户画像
```

---

## 二、选题验证 ⭐⭐⭐⭐⭐

### 检查项

```
□ 三维匹配定义清晰？（人×情境×信息）
□ 选题符合 about-me.md 主题池？
□ 未触犯 about-me.md 禁区？
```

### 成功案例对照

| 维度 | ✅ 成功案例 | ❌ 失败案例 |
|------|------------|------------|
| 三维匹配 | 精准定义：谁+什么情境+什么信息 | 只有模糊的"热门选题" |
| 禁区遵守 | 80%基础+20%进阶 | 全是进阶内容 |
| 差异化 | 有明确的信息增量 | 与现有内容雷同 |

### 问题诊断

```
三维缺失 → 重新定义选题价值
触犯禁区 → 立即更换选题
```

---

## 三、先验调研 ⭐⭐⭐⭐⭐

### 检查项

```
□ 已在小红书搜索？
□ 已在知乎搜索？
□ 已在微信搜一搜？
□ 已有搜索结果数量记录？
□ 已有竞品内容分析？
```

### 市场判断标准

```
搜索结果数量判断：
├─ <10条 → ✅ 金矿，快速介入
├─ 10-100条 → ⚠️ 需要差异化
└─ >100条 → ❌ 建议换主题

质量判断：
□ 现有内容质量如何？
□ 有没有明显的内容缺口？
□ 我的内容能提供什么独特价值？
```

### 成功案例对照

| 维度 | ✅ 成功案例 | ❌ 失败案例 |
|------|------------|------------|
| 调研深度 | 至少3个平台交叉验证 | 只看1个平台 |
| 判断依据 | 有数据支撑（数量+质量） | 凭感觉判断 |
| 结论清晰 | 明确"做"或"不做" | 调研后仍犹豫 |

### 问题诊断

```
无调研 → 立即停止，去做调研
调研<10条 → 金矿，可以做
调研>100条 → 换选题
```

---

## 四、信息增量评估 ⭐⭐⭐⭐

### 检查项

```
□ 新信息占比 ≥70%？
□ 与现有竞品有明确差异化？
□ 提供了新视角/新方法/新洞察？
□ 信息增量可量化描述？
```

### 成功案例对照

| 维度 | ✅ 成功案例 | ❌ 失败案例 |
|------|------------|------------|
| 信息价值 | 70%以上内容是新的 | 大段重复已有内容 |
| 差异化 | 有独特切入角度 | 与竞品内容雷同 |
| 可验证 | 能说出"新在哪里" | 模糊的"更全面" |
