# 发布前检查

> **触发时机**：草稿完成，准备发布前
> **目标**：标题优化、终审打分

---

## 一、标题优化 ⭐⭐⭐⭐⭐

### 检查项

```
□ 有数字对比？（必填）
□ 倍数 ≥10倍？（必填）
□ 有戏剧性词汇？（必填）
□ 小白能理解？（必填）
```

### 数字对比检查

```
✅ 正确示例：
- "效率提升10倍"
- "从3小时→15分钟"
- "100个技巧→3个核心"

❌ 错误示例：
- "效率提升2倍"（<10倍）
- "效率提升一点点"
```

### 戏剧性词汇检查

```
高吸引力词汇库：
- 秘密、真相、颠覆、终极
- 完整、一文搞定、必学
- 踩坑、避坑、血泪教训
- 终极指南、入门手册
```

### 成功案例对照

| 维度 | ✅ 成功案例 | ❌ 失败案例 |
|------|------------|------------|
| 数字 | "3分钟学会"（有具体数字） | "快速学会"（无数字） |
| 倍数 | ≥10倍数量级差异 | <10倍，冲击力不足 |
| 戏剧性 | "终极指南"、"必学" | "XX教程"（平淡） |
| 门槛 | 妈妈能理解 | 专业术语堆砌 |

### 标题模板

```
模板1：从[旧]到[新]
→ 从3小时到15分钟：XX入门指南

模板2：X分钟学会
→ 3分钟学会XX：新手必看

模板3：痛点+方案
→ XX问题？一个方法搞定

模板4：数字+颠覆
→ 100个技巧→3个核心：XX颠覆认知
```

## 三、终审打分 ⭐⭐⭐⭐⭐

### 评分表

```
选题（20分）
├─ 三维匹配清晰        10
├─ 市场洼地（<10条）    5
└─ 信息增量≥70%        5

结构（25分）
├─ "为什么"章节        10
├─ 认知≥3个理由        5
├─ 3-5个场景           5
└─ 覆盖≥3类人群        5

表达（25分）
├─ 承诺具体可信         5
├─ 段落≤5-7行          5
├─ 开头3秒抓住人        5
└─ 术语降维到位         5

视觉（15分）
├─ 抽象概念有图         5
├─ 操作步骤有截图       5
└─ 流程 ASCII 可视化    5

标题（15分）
├─ 有数字对比          5
├─ 倍数≥10倍           5
└─ 戏剧性+低门槛        5
```

### 发布标准

```
≥90分 → 精品，立即发布
80-89分 → 良好，可发布
60-79分 → 优化后发布
<60分 → 重做
```
