# 04-标题

> **核心原则**
>
> 1. **数字冲击**：倍数≥10倍
> 2. **戏剧性词汇**：激发好奇心
> 3. **门槛降低**：小白能理解

---

## 核心公式

```
标题吸引力 = 数字对比 × 戏剧性 × 低门槛
```

**三要素缺一不可**：
- 只有数字 → 缺乏情绪
- 只有戏剧性 → 像标题党
- 门槛太高 → 看不懂

---

## 数字对比

```
大脑处理速度：
数字 <100ms
术语 500-1000ms
描述 1000-2000ms

结论：数字 = 最快触达
```

**倍数要求**：
```
<10倍：冲击力不足
≥10倍：数量级差异，足够震撼
```

**示例**：
```
❌ "效率提升2倍"
✅ "效率提升10倍"
✅ "从3小时→15分钟"
```

---

## 戏剧性词汇

**高吸引力词汇**：
```
- 秘密、真相、颠覆
- 终极、完整、一文搞定
- 踩坑、避坑、血泪教训
```

**示例**：
```
❌ "XX使用教程"
✅ "XX终极指南：3分钟上手"
```

---

## 门槛降低

**原则**：你妈妈能理解吗？

**方法**：
```
1. 避免直接用术语
2. 用结果代替过程
3. 用场景代替概念
```

**示例**：
```
❌ "MCP Server配置教程"
✅ "让AI帮你自动打车"
```

---

## 标题模板

### 模板1：数字对比型
```
从[旧数字]到[新数字]：[主题]
```

### 模板2：时间承诺型
```
[X分钟]学会[主题]
```

### 模板3：问题解决型
```
[痛点]？[方案]帮你搞定
```
