# 01-选题

> **核心原则**
>
> 1. **三维匹配**：价值 = 人×情境×信息
> 2. **先验调研**：不做调研=闭门造车
> 3. **市场洼地**：搜索<10条=早期，>100条=晚期
> 4. **信息增量≥70%**：重复内容=浪费时间

---

## 三维匹配法

```
价值 = 人 (Who) × 情境 (Context) × 信息 (What)

任何一维为0，价值就为0
```

### 维度1：人 (Who)

```
人 = 认知结构 + 技能基础 + 资源约束

- 认知结构：知道什么，不知道什么
- 技能基础：会做什么，不会做什么
- 资源约束：时间/金钱/注意力
```

### 维度2：情境 (Context)

```
情境 = 触发事件 + 当前状态 + 期望状态 + 行动阻碍

- 触发事件：什么事让他需要这个信息
- 当前状态：他现在在哪里
- 期望状态：他想到哪里
- 行动阻碍：什么阻止他到达
```

### 维度3：信息 (What)

```
信息 = 信息增量 + 认知门槛 + 行动成本

- 信息增量：缩短"当前→期望"的距离
- 认知门槛：理解需要多少背景知识
- 行动成本：应用需要多少资源
```

---

## 先验调研

**必做步骤**：

```
1. 搜索：小红书/知乎/微信搜一搜
2. 数量判断：
   - <10条 = 早期市场（金矿）
   - 10-100条 = 中期市场（可做）
   - >100条 = 晚期市场（换主题）
3. 质量判断：现有内容好不好？有没有空间？
```

---

## 市场洼地判断

```
搜索结果数量：
├─ <10条 → 金矿，快速介入
├─ 10-100条 → 中期，需要差异化
└─ >100条 → 晚期，建议换主题
```

---

## 信息增量检查

```
□ 新信息占比 ≥ 70%？
□ 与现有内容有差异化？
□ 提供了新视角/新方法？
```

---

## 检查清单

```
□ 三维匹配定义清晰？
□ 做了先验调研？
□ 市场洼地（搜索<10条）？
□ 信息增量≥70%？
□ 覆盖≥3类人群？
```
