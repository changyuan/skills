# 02-大纲

> **核心原则**
>
> 1. **销售漏斗4阶段**：认知→兴趣→决策→行动
> 2. **承诺设计**：承诺 = 实际价值 × 0.8
> 3. **必须有"为什么"章节**：≥3个理由
> 4. **章节密度**：40-60行/章最佳

---

## 销售漏斗4阶段

```
认知 → 兴趣 → 决策 → 行动
 │      │       │       │
 ↓      ↓       ↓       ↓
为什么  适合我  怎么做   下一步
要用？  吗？           学什么？
```

### 阶段1：认知（必须≥3个理由）

```
用户心理：为什么要用？

回答方式：
- 理由1：[解决什么问题]
- 理由2：[带来什么好处]
- 理由3：[为什么现在要用]

⚠️ 这是最常被跳过的阶段！
```

### 阶段2：兴趣

```
用户心理：适合我吗？

回答方式：
- 场景1："你是不是遇到过..."
- 场景2："当你想要..."
- 场景3："如果你正在..."

目标：3-5个场景，覆盖≥3类人群
```

### 阶段3：决策

```
用户心理：难学吗？

回答方式：
- 详细步骤
- 截图演示
- 强调"简单"、"只需X步"
```

### 阶段4：行动

```
用户心理：接下来学什么？

回答方式：
- 80%基础内容
- 20%进阶内容
- 标注：必学/可选
```

---

## 承诺设计

```
承诺 = 实际价值 × 0.8

原则：稍微低承诺，超额交付
```

### 承诺强度

| 承诺类型 | 强度 | 示例 |
|---------|------|------|
| 强承诺 | 高 | "3分钟学会XX" |
| 中承诺 | 中 | "XX完整指南" |
| 弱承诺 | 低 | "聊聊XX" |

---

## 章节密度

```
最佳密度：40-60行/章

<40行/章：内容太稀疏
>60行/章：信息过载
```

---

## 检查清单

```
□ 有"为什么"章节？
□ 认知阶段≥3个理由？
□ 3-5个场景？
□ 覆盖≥3类人群？
□ 承诺：具体+可信+有吸引力？
□ ≤15章？
□ 40-60行/章？
```
