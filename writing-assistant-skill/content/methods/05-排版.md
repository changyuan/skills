# 05-排版

> **核心原则**
>
> 1. **三级标题**：只用H1/H2/H3
> 2. **公众号适配**：有序列表无空格
> 3. **换行标记**：用 `---` 换行
> 4. **流程可视化**：用 ASCII 图

---

## 三级标题系统

```
只用3级标题：H1 / H2 / H3

四级以上 → 层级混乱
```

**标题规范**：

| 标题 | 用途 |
|------|------|
| H1 | 主要章节 |
| H2 | 子章节 |
| H3 | 小节 |

---

## 公众号适配

### 有序列表格式

```
✅ 正确：
1.项目一（数字后直接接内容）
2.项目二

❌ 错误：
1. 项目一（数字后有空格）
2. 项目二
```

### 换行标记

```
用 `---` 产生大间距

段落1

---

段落2
```

---

## ASCII 可视化

**流程/步骤必须用 ASCII 图**：

```
步骤1 → 步骤2 → 步骤3
```

**复杂流程**：
```
┌─────────┐    ┌─────────┐
│  输入   │ →  │  输出   │
└─────────┘    └─────────┘
```

**循环流程**：
```
    ┌──────────────┐
    │              │
    ↓              │
  步骤1 → 步骤2 → 步骤3
```
