## 三、问题诊断清单 ⭐⭐⭐⭐⭐

### 3.1 阅读量低 → 标题/封面问题

```
检查项：
□ 标题数字对比够不够强？
□ 倍数是否 ≥10倍？
□ 戏剧性词汇是否到位？
□ 小白能否看懂？
□ 封面是否吸引点击？

失败教训：
- 标题太专业，小白不敢点
- 数字太小，冲击力不足
- 缺少情绪词，太平淡
```

### 3.2 点赞率低 → 内容价值问题

```
检查项：
□ 开头3秒是否抓住人？
□ 是否有"为什么"章节？
□ 认知阶段是否≥3个理由？
□ 信息增量是否足够？
□ 与竞品差异是否明显？

失败教训：
- 开头太慢，用户流失
- 只有"怎么做"，没有"为什么"
- 内容与标题承诺不符
```

### 3.3 收藏率低 → 实用价值问题

```
检查项：
□ 是否提供了可复用的方法/模板？
□ 步骤是否清晰可操作？
□ 是否有独特的洞察/视角？
□ 是否标注了重点和优先级？

失败教训：
- 只有道理，没有方法
- 操作步骤太抽象，无法落地
- 内容太泛，缺乏独特价值
```

### 3.4 转发率低 → 社交货币问题

```
检查项：
□ 是否能体现转发者的认知/品味？
□ 是否有值得讨论的观点？
□ 是否能帮助转发者表达立场？
□ 是否有戏剧性/反常识内容？

失败教训：
- 内容太"正确"，无法体现独特性
- 缺乏可讨论的观点
- 转发没有社交价值
```

### 3.5 完读率低 → 开篇/节奏问题

```
检查项：
□ 开头3秒是否有痛点/对比/反常识？
□ 承诺是否足够吸引？
□ 段落是否太长（>7行）？
□ 章节密度是否合适（40-60行/章）？
□ 是否有视觉元素打断阅读疲劳？

失败教训：
- 开头铺垫太长
- 段落密集成"墙"
- 缺乏图片/图示调节
```
