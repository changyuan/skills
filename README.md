# skills

A collection of skills and tools for various purposes.

## Directories

- **[media-poster/](./media-poster/)** - Media poster tools and utilities
- **[media-poster-script/](./media-poster-script/)** - Script-based evaluation harness for media-poster MCP server. Provides tool calling, response parsing, and automated testing without requiring MCP client sessions.
- **[mimeng-writing/](./mimeng-writing/)** - 咪蒙爆款文章写作技巧。适用于需要创作10万+阅读量爆款文章、情感共鸣类内容、故事叙事或社会议题评论时使用。掌握标题制造、开篇设计、情绪调动、金句提炼、故事叙事等核心技巧。
- **[writing-assistant-skill/](./writing-assistant-skill/)** - 自媒体写作助手 Skill，帮助完成从选题到发布的完整流程。包含选题、大纲、正文、标题、排版等核心方法论，以及完整的写作前、中、后检查流程。
